# Twitter-clone

Created a twitter clone using MERN stack

### Features

-   Tweet only text
-   Tweet both text and image
-   Delete the tweet
-   Covid-19 api included which shows all the data about different states in India.
-   Based on material design.

### Technology used

-   React for frontend.
-   Node.js for making api in the backend
-   Express.js backend framework.
-   MongoDB NoSQL database with cloud storage atlas server.
-   Firebase storage.

-   Read [.env_sample](https://github.com/sambeetpanda507/twitter-clone/blob/master/server/.env_sample) and make .env inside [server](https://github.com/sambeetpanda507/twitter-clone/tree/master/server) and add the required parameters.
-   Read [.firebase_sample](https://github.com/sambeetpanda507/twitter-clone/blob/master/client/src/firebase_sample.js) and create a .fiebase file inside [src](https://github.com/sambeetpanda507/twitter-clone/tree/master/client/src) and add the required parameters.
